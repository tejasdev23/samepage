# Samepage
Real-time Synchronized Media Player  

Tech Stack: Java 21, Spring Boot 3.4, WebSockets, Redis. 

Key Feature: Full-duplex synchronization of video state (Play/Pause/Seek) across distributed clients using a event-driven architecture
